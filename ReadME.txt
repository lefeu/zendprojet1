Ce projet a pour but de gérer une collection d'albums de musique.

Sur L'application web , l'uitlisateur doit pouvoir ajouter,modifier, supprimer un album .

Pour l'execution du projet :

##########  Pré-requis ########
  Bien installer php, un serveur web, zend 

######## Installer la base de données ###########
  1- créer la base mysql de nom musicotheque 
  2- importer le fichier musicotheque.sql dans votre Base . Ce fichier se trouve dans le repertoire BD
  3- Aller dans le repertoire configs , ouvrir le fichier aaplication.ini et 
     changer les  informations resources.db.params.username  et resources.db.params.password si elles ne sont pas les meme 
     que les votres (ce sont des informations de connexion à la base de données MYSQL . )


#######  Lancer l'url suivant ###########

  http://localhost/tp-zf/public/
